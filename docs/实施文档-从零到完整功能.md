# Conduit 实施文档 — 从零到完整功能

> 版本：2026-06-18　|　配套阅读：《调研-Conduit终端方案.md》《调研-三大难点深入.md》
>
> **目标**：照本文从空目录搭到完整功能。涵盖环境、脚手架、目录结构、每个 Rust 命令、统一 Agent Harness（含完整代码）、Git 集成、前端接线、颜值（macOS 毛玻璃）、分阶段里程碑与验收清单。
>
> **已固化决策**（详见三大难点文档·决策记录）：
> - **D1 "应用补丁"走简单风**：agent 直接写盘，UI 只展示结果 + diff（+ 写盘护栏 + 「丢弃改动」）。
> - **D2 只接 CC + CX 两家**（砍掉 Cursor/CU）；二者均支持 resume。
> - **D3 统一调 CLI**（claude / codex），不引入 SDK。
> - **D4 commit + push 全走系统 `git` CLI**（禁交互+超时+stderr 回显）；status/diff/ahead-behind 用 `git2`。
> - **D5 终态（status/result/error/updatedAt）MVP 必持久化**；逐字 transcript 重放二期。
> - **D6 并发软上限可配**（默认 ≤4，超限**拒绝+提示**，排队二期）。

### 3.3 ★新增：统一 Agent Harness

**3.3.1 归一化事件（`agent/event.rs`）**
```rust
use serde::Serialize;

#[derive(Serialize, Clone)]
#[serde(tag = "kind", rename_all = "camelCase")]
pub enum AgentEvent {
    /// 会话开始；session_id 供多轮 resume（CC=session_id / CX=thread_id）
    Started { agent_session_id: Option<String> },
    /// 增量文本（打字机效果）
    Delta { text: String },
    /// 工具调用（UI 折叠展示）
    ToolUse { name: String, summary: Option<String> },
    /// 文件改动 → 触发 git diff 刷新
    FileChange { path: String },
    /// 完成
    Done { ok: bool, result: Option<String>, cost_usd: Option<f64> },
    /// 失败
    Failed { message: String },
}
```

**3.3.2 Harness trait + 三实现（`agent/harness.rs`）**

每个 harness 负责：① 构造 CLI 命令（含 §决策 D1 的直接写盘 flag）；② 把该 CLI 的原生 JSONL 行解析成 `AgentEvent`。

```rust
use crate::agent::event::AgentEvent;
use serde_json::Value;

pub enum AgentKind { Claude, Codex }   // CU(Cursor) 已砍（D2）

impl AgentKind {
    pub fn from_code(c: &str) -> Option<Self> {
        match c { "CC" => Some(Self::Claude), "CX" => Some(Self::Codex), _ => None }
    }
    pub fn bin(&self) -> &'static str {
        match self { Self::Claude=>"claude", Self::Codex=>"codex" }
    }

    /// 构造命令行参数（D1：均直接写盘）
    pub fn args(&self, prompt: &str, resume: Option<&str>) -> Vec<String> {
        match self {
            // CC: stream-json + 自动接受编辑 + 可 resume
            Self::Claude => {
                let mut a = vec![
                    "-p".into(), prompt.into(),
                    "--output-format".into(), "stream-json".into(),
                    "--verbose".into(), "--include-partial-messages".into(),
                    "--permission-mode".into(), "acceptEdits".into(),  // D1 直接写盘
                ];
                if let Some(sid) = resume { a.push("--resume".into()); a.push(sid.into()); }
                a
            }
            // CX: exec --json + workspace-write；resume 用子命令形式（见 spawn 注）
            Self::Codex => {
                let mut a = vec!["exec".into()];
                if let Some(tid) = resume { a.push("resume".into()); a.push(tid.into()); }
                a.extend([
                    prompt.into(),
                    "--json".into(),
                    "--sandbox".into(), "workspace-write".into(),  // D1 直接写盘
                    "--skip-git-repo-check".into(),                // P1：非 git 目录也能跑
                ]);
                a
            }
        }
    }

    /// 解析一行原生 JSONL → 归一化事件（None = 忽略该行）
    /// ⚠️ 字段名需对照 §1.5 钉定版本的 golden fixtures 校正；缺字段返回 None 不 panic。
    pub fn parse_line(&self, line: &str) -> Option<AgentEvent> {
        let v: Value = serde_json::from_str(line).ok()?;
        match self {
            Self::Claude => match v.get("type")?.as_str()? {
                "system" if v.get("subtype")?.as_str()? == "init" =>
                    Some(AgentEvent::Started { agent_session_id: v.get("session_id").and_then(|s| s.as_str()).map(String::from) }),
                "stream_event" => {
                    let d = v.get("event")?.get("delta")?;
                    if d.get("type")?.as_str()? == "text_delta" {
                        Some(AgentEvent::Delta { text: d.get("text")?.as_str()?.into() })
                    } else { None }
                }
                "result" => {
                    let ok = !v.get("is_error").and_then(|b| b.as_bool()).unwrap_or(false)
                             && v.get("subtype").and_then(|s| s.as_str()) == Some("success");
                    Some(AgentEvent::Done {
                        ok,
                        result: v.get("result").and_then(|s| s.as_str()).map(String::from),
                        cost_usd: v.get("total_cost_usd").and_then(|n| n.as_f64()),
                    })
                }
                _ => None,
            },
            Self::Codex => match v.get("type")?.as_str()? {
                "thread.started" =>
                    Some(AgentEvent::Started { agent_session_id: v.get("thread_id").and_then(|s| s.as_str()).map(String::from) }),
                "item.completed" => {
                    // file_change → FileChange；agent_message 文本 → Delta（CX 通常整段给）
                    let item = v.get("item")?;
                    match item.get("type")?.as_str()? {
                        "file_change" => Some(AgentEvent::FileChange {
                            path: item.get("path").and_then(|s| s.as_str()).unwrap_or("").into() }),
                        "agent_message" => Some(AgentEvent::Delta {
                            text: item.get("text").and_then(|s| s.as_str()).unwrap_or("").into() }),
                        _ => None,
                    }
                }
                "turn.completed" => Some(AgentEvent::Done { ok: true, result: None, cost_usd: None }),
                "turn.failed" => Some(AgentEvent::Failed { message: "turn failed".into() }),
                _ => None,
            },
        }
    }
}
```

**3.3.3 进程管理 + 流式推送（`agent/mod.rs`）**
```rust
use std::collections::HashMap;
use std::sync::atomic::{AtomicU32, AtomicUsize, Ordering};
use std::sync::{Arc, RwLock};
use tauri::ipc::Channel;
use tokio::io::{AsyncBufReadExt, BufReader};
use tokio::process::{Child, Command};

use crate::agent::{event::AgentEvent, harness::AgentKind};

// D6：软上限可配。默认 4，运行时从设置（tauri-plugin-store）读，存进 AgentState。
const DEFAULT_MAX_CONCURRENT: usize = 4;

/// 每个在跑的 agent：持有取消句柄（kill 子进程）。child 本体归读取任务所有，
/// 任务在子进程退出后负责从 procs 自我清理（修 P0：完成不移除→永久占名额）。
struct RunningAgent { killer: tokio::sync::Mutex<Option<tokio::process::Child>> }

type Procs = Arc<RwLock<HashMap<u32, Arc<RunningAgent>>>>;

pub struct AgentState {
    procs: Procs,                  // Arc 以便后台任务持有并自清理
    next_id: AtomicU32,
    max_concurrent: AtomicUsize,   // 可由设置项调整
}
impl Default for AgentState {
    fn default() -> Self {
        Self { procs: Arc::new(RwLock::new(HashMap::new())), next_id: AtomicU32::new(0),
               max_concurrent: AtomicUsize::new(DEFAULT_MAX_CONCURRENT) }
    }
}
impl AgentState { fn clone_handle(&self) -> Procs { self.procs.clone() } }

#[tauri::command]
pub async fn agent_spawn(
    state: tauri::State<'_, AgentState>,
    agent: String,          // "CC"|"CX"
    prompt: String,
    cwd: String,
    resume: Option<String>, // agent_session_id（CC=session_id / CX=thread_id）
    on_event: Channel<AgentEvent>,
) -> Result<u32, String> {
    // D6：MVP 为“超限拒绝 + 提示”（非排队，见 §9 P1）。
    if state.procs.read().unwrap().len() >= state.max_concurrent.load(Ordering::Relaxed) {
        return Err("并发上限：请等待其他会话完成或在设置中调高上限".into());
    }
    let kind = AgentKind::from_code(&agent).ok_or("未知 agent")?;
    let mut child = Command::new(kind.bin())
        .args(kind.args(&prompt, resume.as_deref()))
        .current_dir(&cwd)
        .stdout(std::process::Stdio::piped())
        .stderr(std::process::Stdio::piped())   // 必须 drain，见下（修 P0：写满管道会 hang）
        .kill_on_drop(true)
        .spawn()
        .map_err(|e| format!("启动 {} 失败：{e}（是否已安装/登录？）", kind.bin()))?;

    let stdout = child.stdout.take().ok_or("no stdout")?;
    let stderr = child.stderr.take().ok_or("no stderr")?;
    let id = state.next_id.fetch_add(1, Ordering::Relaxed) + 1;
    // child 交给读取任务管理；procs 只持取消句柄
    let running = Arc::new(RunningAgent { killer: tokio::sync::Mutex::new(Some(child)) });
    state.procs.write().unwrap().insert(id, running.clone());

    // 修 P0：drain stderr（攒成失败诊断；否则管道写满整个 agent 卡死）
    let stderr_buf = Arc::new(tokio::sync::Mutex::new(String::new()));
    {
        let buf = stderr_buf.clone();
        tokio::spawn(async move {
            let mut lines = BufReader::new(stderr).lines();
            while let Ok(Some(l)) = lines.next_line().await {
                let mut g = buf.lock().await; g.push_str(&l); g.push('\n');
            }
        });
    }

    let ch = on_event.clone();
    let procs = state.procs.clone_handle();   // 见下：AgentState 暴露 Arc 克隆以便任务自清理
    tokio::spawn(async move {
        // 逐行读 stdout → 解析 → 推前端
        let mut lines = BufReader::new(stdout).lines();
        let mut saw_done = false;
        while let Ok(Some(line)) = lines.next_line().await {
            if let Some(ev) = kind.parse_line(&line) {
                if matches!(ev, AgentEvent::Done{..} | AgentEvent::Failed{..}) { saw_done = true; }
                let _ = ch.send(ev);
            }
        }
        // 修 P0：stdout EOF 后 await 真实退出码（try_wait 可能尚未就绪）
        let status = {
            let mut g = running.killer.lock().await;
            match g.as_mut() { Some(c) => c.wait().await.ok(), None => None }
        };
        let ok = status.map(|s| s.success()).unwrap_or(false);
        // 兜底：CC 偶发 result 缺失/hang（issue #1920/#8126/#25629）——没见过 Done 才补发
        if !saw_done {
            let msg = stderr_buf.lock().await.clone();
            if ok { let _ = ch.send(AgentEvent::Done { ok: true, result: None, cost_usd: None }); }
            else  { let _ = ch.send(AgentEvent::Failed { message:
                        if msg.is_empty() { "agent 异常退出".into() } else { msg } }); }
        }
        // 修 P0：无论如何从 procs 移除，释放并发名额
        procs.write().unwrap().remove(&id);
    });
    Ok(id)
}

#[tauri::command]
pub async fn agent_cancel(state: tauri::State<'_, AgentState>, id: u32) -> Result<(), String> {
    // 取句柄并 kill；读取任务会在 wait 返回后自行 remove
    let running = state.procs.read().unwrap().get(&id).cloned();
    if let Some(r) = running {
        if let Some(c) = r.killer.lock().await.as_mut() { let _ = c.start_kill(); }
    }
    Ok(())
}

// 注：AgentState 需提供 `clone_handle()` 返回 `Arc<RwLock<HashMap<u32, Arc<RunningAgent>>>>`，
//     使后台任务能在子进程退出后自清理 procs（把 procs 字段本身设为 Arc 即可）。
// agent_write（向 agent stdin 追加多轮输入）按需实现；多轮主路径用 resume 重 spawn。
```

**3.3.4 预检（`agent/preflight.rs`）**
三层预检（P1）：① 可执行存在 → ② 登录态 → ③（可选）版本范围。
```rust
use serde::Serialize;
#[derive(Serialize)]
pub struct Preflight { pub installed: bool, pub logged_in: bool, pub hint: Option<String> }

#[tauri::command]
pub async fn agent_preflight(agent: String) -> Result<Preflight, String> {
    let kind = crate::agent::harness::AgentKind::from_code(&agent).ok_or("未知 agent")?;
    let bin = kind.bin();
    // ① 可执行存在
    if which::which(bin).is_err() {
        return Ok(Preflight { installed: false, logged_in: false,
            hint: Some(format!("未找到 {bin}，请先安装该 agent CLI")) });
    }
    // ② 登录态：各 CLI 命令不同；命令缺失/异常则降级为“未知”，由 spawn 时再报
    let logged_in = match kind {
        crate::agent::harness::AgentKind::Claude =>
            run_ok(bin, &["auth", "status"]).await,        // claude auth status
        crate::agent::harness::AgentKind::Codex =>
            run_ok(bin, &["login", "status"]).await,       // codex login status（有凭证退出 0）
    };
    // ③ 版本范围：可选，对照 §1.5 钉定版本给兼容性提示（此处略）
    Ok(Preflight { installed: true, logged_in,
        hint: if logged_in { None } else { Some(format!("{bin} 似乎未登录，请先登录")) } })
}
```

### 3.4 ★新增：Git 集成

**3.4.1 读：status / diff / ahead-behind（`git/mod.rs`，用 git2）**

> 修 P1：要覆盖 **staged(HEAD→index) + unstaged(index→workdir) + untracked**，并给出**每文件** `+x −y`，不留 `0/0` 占位。做法：对 workdir 做一次 `diff_tree_to_workdir_with_index`（含 HEAD tree 比对 + untracked），用 per-file patch 累计行数。

```rust
use std::collections::HashMap;
use git2::{Repository, DiffOptions};
use serde::Serialize;

#[derive(Serialize)]
pub struct FileChange { pub path: String, pub status: String, pub added: usize, pub removed: usize }
#[derive(Serialize)]
pub struct StatusResult { pub branch: String, pub files: Vec<FileChange>, pub summary: String }

#[tauri::command]
pub fn git_status(repo_path: String) -> Result<StatusResult, String> {
    let repo = Repository::discover(&repo_path).map_err(|e| e.to_string())?;
    let branch = repo.head().ok()
        .and_then(|h| h.shorthand().map(String::from))
        .unwrap_or_else(|| "HEAD".into());

    // HEAD tree → workdir（经 index），含 untracked，覆盖 staged+unstaged 全部改动
    let head_tree = repo.head().ok()
        .and_then(|h| h.peel_to_tree().ok());
    let mut opts = DiffOptions::new();
    opts.include_untracked(true).recurse_untracked_dirs(true);
    let diff = repo.diff_tree_to_workdir_with_index(head_tree.as_ref(), Some(&mut opts))
        .map_err(|e| e.to_string())?;

    // 每文件累计 +/−：遍历 patch 的 line origin
    let mut per_file: HashMap<String, (usize, usize, char)> = HashMap::new();
    diff.print(git2::DiffFormat::Patch, |delta, _h, line| {
        let path = delta.new_file().path().or_else(|| delta.old_file().path())
            .map(|p| p.to_string_lossy().into_owned()).unwrap_or_default();
        let mark = match delta.status() {
            git2::Delta::Added | git2::Delta::Untracked => 'A',
            git2::Delta::Deleted => 'D',
            _ => 'M',
        };
        let e = per_file.entry(path).or_insert((0, 0, mark));
        match line.origin() { '+' => e.0 += 1, '-' => e.1 += 1, _ => {} }
        true
    }).map_err(|e| e.to_string())?;

    let mut files: Vec<FileChange> = per_file.into_iter()
        .map(|(path, (added, removed, mark))| FileChange { path, status: mark.to_string(), added, removed })
        .collect();
    files.sort_by(|a, b| a.path.cmp(&b.path));
    let (ta, tr): (usize, usize) = files.iter().fold((0, 0), |(a, r), f| (a + f.added, r + f.removed));
    let summary = format!("{} 文件 · +{} −{}", files.len(), ta, tr);
    Ok(StatusResult { branch, files, summary })
}

#[tauri::command]
pub fn git_diff(repo_path: String, file: String) -> Result<String, String> {
    let repo = Repository::discover(&repo_path).map_err(|e| e.to_string())?;
    let head_tree = repo.head().ok().and_then(|h| h.peel_to_tree().ok());
    let mut opts = DiffOptions::new();
    opts.include_untracked(true).pathspec(&file);
    let diff = repo.diff_tree_to_workdir_with_index(head_tree.as_ref(), Some(&mut opts))
        .map_err(|e| e.to_string())?;
    let mut out = String::new();
    diff.print(git2::DiffFormat::Patch, |_d, _h, l| {
        let origin = l.origin();   // '+' '-' ' '
        if matches!(origin, '+'|'-'|' ') { out.push(origin); }
        out.push_str(std::str::from_utf8(l.content()).unwrap_or(""));
        true
    }).map_err(|e| e.to_string())?;
    Ok(out)   // 前端把 +/− 行染成设计稿 diff 配色
}

#[tauri::command]
pub fn git_ahead_behind(repo_path: String) -> Result<(usize, usize), String> {
    let repo = Repository::discover(&repo_path).map_err(|e| e.to_string())?;
    let head = repo.head().map_err(|e| e.to_string())?;
    let local = head.target().ok_or("no head oid")?;
    let branch = head.shorthand().ok_or("no branch")?;
    let upstream = repo.find_branch(branch, git2::BranchType::Local)
        .and_then(|b| b.upstream()).map_err(|e| e.to_string())?;
    let up_oid = upstream.get().target().ok_or("no upstream oid")?;
    repo.graph_ahead_behind(local, up_oid).map_err(|e| e.to_string())  // (ahead, behind)
}
```

**3.4.2 写：commit + push 全走系统 `git` CLI（D4 + P2-7）（`git/commit.rs`）**
```rust
use std::time::Duration;
use tokio::process::Command;
use tokio::time::timeout;

const GIT_TIMEOUT: Duration = Duration::from_secs(60);

/// 统一跑 git 子命令：禁交互（不挂起等密码）+ 超时 + 捕获 stderr 回 UI。
async fn run_git(repo_path: &str, args: &[&str]) -> Result<String, String> {
    let fut = Command::new("git")
        .current_dir(repo_path)
        .args(args)
        .env("GIT_TERMINAL_PROMPT", "0")    // P2-7：要凭证时立即失败而非挂起
        // 如需进一步禁 ssh 交互：.env("GIT_SSH_COMMAND", "ssh -oBatchMode=yes")
        .output();
    let out = match timeout(GIT_TIMEOUT, fut).await {
        Err(_) => return Err("git 超时（可能在等待凭证/网络）。请先在终端完成登录或凭证配置。".into()),
        Ok(Err(e)) => return Err(format!("无法执行 git：{e}（是否已安装 git？）")),
        Ok(Ok(o)) => o,
    };
    if out.status.success() {
        Ok(String::from_utf8_lossy(&out.stdout).into_owned())
    } else {
        // stderr 原样回显（如 "Authentication failed" / "could not read Username"）
        Err(String::from_utf8_lossy(&out.stderr).into_owned())
    }
}

#[tauri::command]
pub async fn git_commit(repo_path: String, message: String) -> Result<String, String> {
    // 暂存全部改动后提交。`git commit` 自动继承 user.name/email、pre-commit hooks、GPG 签名。
    run_git(&repo_path, &["add", "-A"]).await?;
    run_git(&repo_path, &["commit", "-m", &message]).await
    // 失败场景由 run_git 回显：无改动可提交、pre-commit hook 拒绝等。
}

#[tauri::command]
pub async fn git_push(repo_path: String) -> Result<(), String> {
    run_git(&repo_path, &["push"]).await.map(|_| ())
    // 凭证复用系统 credential helper / ssh-agent；需登录时 stderr 提示回 UI。
}
```

### 4.2 会话编排 store（`state/sessions.ts`）
```ts
export type AgentCode = 'CC'|'CX';                 // CU 已移除（D2）
export type SessionStatus = 'running'|'fresh'|'failed'|'done';
export type SessionKind = 'shell'|'agent';

// 内联块：少量结构（toolUse/fileChange/轮次分隔）。正文 delta 不进数组，见 reply。
export type AgentBlock =
  | { type:'toolUse'; name:string; summary?:string }
  | { type:'fileChange'; path:string }
  | { type:'turn' };

export type Session = {
  id: string;
  kind: SessionKind;
  agent?: AgentCode;
  agentSessionId?: string;          // CC=session_id / CX=thread_id（多轮 resume）
  procId?: number;                  // 后端 agent_spawn / pty_open 返回的 id
  title: string;
  dir: string; branch: string;
  status: SessionStatus;
  // ⚠️ 不要把每条 delta 存成数组（长会话 O(n²) 复制）。正文累积进字符串：
  reply: string;                    // 打字机正文（仅追加）
  blocks: AgentBlock[];             // 少量块结构
  result?: string; error?: string; updatedAt: number;  // 终态（D5：必持久化）
  changes?: { files: FileChange[]; summary: string; commit?: string };
};

// 分组：按 dir 归组 → 设计稿侧边栏 ~/orbit ~/web ~/infra
export const groupByDir = (s: Session[]) =>
  s.reduce<Record<string, Session[]>>((acc, x) => ((acc[x.dir] ??= []).push(x), acc), {});

// 状态机（被 agent 事件驱动）。注意：delta 只 += 字符串，不复制数组。
export function applyEvent(s: Session, ev: AgentEvent): Session {
  const now = Date.now();
  switch (ev.kind) {
    case 'started':   return { ...s, status:'running', updatedAt:now,
                               agentSessionId: ev.agentSessionId ?? s.agentSessionId };
    case 'delta':     return { ...s, reply: s.reply + ev.text, updatedAt:now };   // 字符串追加，O(1) 摊还
    case 'toolUse':   return { ...s, blocks:[...s.blocks, {type:'toolUse', name:ev.name, summary:ev.summary}], updatedAt:now };
    case 'fileChange':return { ...s, blocks:[...s.blocks, {type:'fileChange', path:ev.path}], updatedAt:now }; // + 去抖刷 git（§4.4）
    case 'done':      return { ...s, status: ev.ok ? 'fresh':'failed', result: ev.result, updatedAt:now };
    case 'failed':    return { ...s, status:'failed', error: ev.message, updatedAt:now };
  }
}
// fresh → done：用户切走 或 N 秒后降级（UI 层定时）。
// 终态持久化：done/failed 时把 {status,result,error,updatedAt} 写入 tauri-plugin-store（D5）。
```

## 5. 关键数据流（端到端）

```
新建 Agent:
  UI 选目录+agent → preflightAgent()(存在/登录) → spawnAgent(agent,prompt,cwd,resume?)
   → [Rust] Command::new(bin).args(...).spawn → tokio 逐行读 stdout（+drain stderr）
   → parse_line → AgentEvent → Channel.send → [UI] onEvent → applyEvent(状态机)
   → reply(字符串)/blocks 更新内联块；done→fresh(绿勾)/failed→通知（终态落 store）
   → fileChange→去抖刷 git
   → [Rust] stdout EOF 后 await child.wait()，从 procs 移除（释放并发名额）

真实终端（复用）:
  term.onData → pty_write ;  pty Data 事件(base64) → term.write ;  resize → pty_resize

Git 面板:
  会话激活 / 窗口 focus / fileChange(去抖) / commit 后 → git_status + git_ahead_behind → 渲染文件卡/领先落后
  点文件 → git_diff(file) → mini diff 染色
  提交 → git_commit(msg)[CLI: add -A + commit] ; 提交并推送 → git_commit + git_push[CLI]
```

## 8. 风险与对策速查

| 风险 | 对策 |
|---|---|
| CLI 字段/版本漂移 | `parse_line` 容错（缺字段返回 None 不 panic）；钉版本 + golden fixtures 回归（§1.5） |
| CC result 事件缺失/hang | stdout 关闭后 await `wait()` 拿真实退出码兜底（§3.3.3） |
| agent stderr 写满管道 hang | drain stderr 任务（§3.3.3） |
| agent 完成不释放并发名额 | 读取任务在 `wait()` 后自清理 procs（§3.3.3） |
| git commit/push 阻塞或认证失败 | CLI + `GIT_TERMINAL_PROMPT=0` + 超时 + stderr 回显（§3.4.2 / D4） |
| 大仓 git 慢 | diff 刷新去抖 300ms + 后台执行；多触发点而非每次按键 |
| agent 跑爆机器 | 并发软上限可配（默认 4）+ 超限拒绝提示（D6） |
| 长会话内存膨胀 | 正文 reply 字符串累积，不存事件数组（避免 O(n²)） |
| terax 复用的 license | 保留其 Apache-2.0 LICENSE 与版权头；新增代码独立 |
